This website runs entirely in your browser using the Bluesky AppView API, with no data sent anywhere. No analytics or trackers are used, so you can explore freely. All JavaScript was developed with the help of LLMs like Mistral AI, Claude, and Google AI Studio. If you want to browse the Bluesky way, here are the links. You can pin it to your home.

New Horizons

Aggregating Real-Time News from BlueSky Media Profiles, Organizations, Institutions, and Journalists.

You can fork this project and create your own version, adding different feeds and lists. The code is available on GitHub; all you need are the lists and feeds. Please note, the entire JavaScript is created with the help of LLMs.

All post data comes from BlueSky, where the accounts on the list have posted it. This project simply aggregates and visualizes that data in a news media format. It's part of my ongoing experience with the BlueSky API.
