Streams news and posts straight from BlueSky

This website runs entirely in your browser using the Bluesky AppView API, with no data sent anywhere. No analytics or trackers are used, so you can explore freely. All JavaScript was developed with the help of LLMs like Mistral AI, Claude, and Google AI Studio. If you want to browse the Bluesky way, here are the links. You can pin it to your home. As more companies, institutions, and organizations join Bluesky, you can use it as a one stop solution for all kinds of updates.

You can pin the list in the Bluesky app for direct access to news, free from algorithms. If you choose the feed, it follows HN Ranking from the past 12 hours.

New Horizons

Aggregating real-time news from Bluesky media profiles, tech companies, organizations, institutions, and journalists.

You can fork this project and create your own version, adding different feeds and lists. The code is available on GitHub; all you need are the lists and feeds. Please note, the entire JavaScript is created with the help of LLMs.

All post data comes from BlueSky, where the accounts on the list have posted it. This project simply aggregates and visualizes that data in a news media format. It's part of my ongoing experience with the BlueSky API.
