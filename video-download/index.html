This tool is for personal use only. Bluesky is an open platform, meaning posts are public and accessible to anyone, even without an account. If you prefer not to download videos, you can stream them using VLC: copy the playlist URL below, then in VLC, go to Media > Stream > Network, and paste the link to play. When downloading, always credit the original author and respect their privacy preferences as stated in their profile or posts. This tool does not store or track your data; how you use it is your responsibility.

Bluesky videos may be available in multiple resolutions. To download a video, you'll need to select your preferred resolution. Bluesky utilizes M3U8 playlists for streaming. An M3U8 file is a plain text file that lists media files for playback. Each post has a unique playlist URL. If you paste this URL into a browser, a playlist file will download, containing details like resolution and other data. To copy the playlist URL, simply click on it.
